¿Que son los iteradores?
Un iterador es un puntero que se utiliza en un algoritmo para poder recorrer los elementos almacenados en un contenedor.
Esto debido que se necesita realizar operaciones en las cuales se requiere del recorrido de los contenedores.

¿Como podriamos recorrer un coleccion en java sin la ayuda de un iterador?
Podemos recorrer una coleccion utilizando ForEach la cual nos permite recorrer una lista de una forma mas compacta 
y reduciendo el codigo.
